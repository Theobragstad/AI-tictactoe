# ❌ Tic-tac-toe ⭕

### Game modes:
Normal, random, Minimax AI, Alpha-beta AI.

### Other features: 
Custom icons, custom board size, multi-game simulations, AI depth settings.

#### How to run:  
- From the main directory, run `python3 game.py`
